# Synapse Web

A single-page, offline-friendly quiz runner (PWA) matching the behaviour of your Tkinter Synapse app.

## Hosting for free (recommended): GitHub Pages

1. Create a new GitHub repository (e.g. `synapse-web`).
2. Upload the contents of this folder to the repo root.
3. In GitHub: **Settings → Pages**
   - Source: **Deploy from a branch**
   - Branch: **main** (root)
4. Your app will be available at: `https://<username>.github.io/<repo>/`

## Install on iPhone/iPad

Open the GitHub Pages URL in Safari → Share → **Add to Home Screen**.

## Notes

- No server required (all runs client-side).
- Your question text is never uploaded anywhere.
